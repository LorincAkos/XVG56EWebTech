SZÓKIRAKÓ JÁTÉK

Adott egy 4*9-es négyzetháló. A program megad egy - a négyzetháló feletti képhez
kapcsolódó - szót vagy mondatot, majd ennek betűit összekeverve elhelyezi ebben a
négyzethálóban.

A játékos feladata az, hogy az összekevertbetűkből kiírja a szót
vagy mondatot a kijelölt helyen. A bekerült vízszintes vagy függőleges irányban egyenként
lehet mozgatni.

Ha a betű útját elállja egy másiik betű, akkor csak a másik betűig tolódik el.

Miután sikerült a betűket a megfelelő helyre mozgatni, a program ezt jelzi, ha viszont valamely

betűket felcseréltük, akkor azokra színkiemeléssel hívja fel a
figylemünket.

Derékszögű háromszög igaz pitagorasz tétele:
a2 + b2 = c2

Általános háromszögek esetén igaz:
α + β + γ = 180°
